# 3d-mesh-beliche

Gera a volumetria 3D de uma beliche (cama principal + degraus/nichos) com `trimesh`
e exporta a malha combinada para `beliche.stl` e `beliche.dae` no diretório atual.

> Código em português (comentários e nomes como `cama`, `degrau`, `beliche`).

## Rodar

```bash
uv run main.py
```

Os arquivos `beliche.stl` e `beliche.dae` são escritos no diretório de onde você roda.

## Testar

```bash
uv run pytest
```
