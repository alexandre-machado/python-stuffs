# multiprocessing-examples

Exemplos de como paralelizar trabalho com `multiprocessing.Pool` e acompanhar o
progresso com `tqdm`. Cobre os padrões `imap`, `imap_unordered` e `apply_async`,
incluindo funções com um argumento, múltiplos argumentos e `functools.partial`.

## Rodar

```bash
uv run main.py
```

## Testar

```bash
uv run pytest
```
